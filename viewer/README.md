# عارض بيانات مصحف ورش المثمن

أداة ويب خفيفة ومستقلة (مبنية بـ HTML/CSS/JS) صُممت خصيصاً لمطوري المشروع والباحثين. تهدف الأداة إلى معاينة صفحات "مصحف ورش المثمن" والتحقق البصري المباشر من صحة ودقة البيانات المستخرجة (مثل إحداثيات الآيات وعلامات الوقف).

## 🌟 المميزات

- **تصميم نقي وخفيف:** واجهة داكنة مريحة للعين تعرض الصفحة بأكملها بدون مشتتات.
- **التنقل السريع:** استخدام أسهم لوحة المفاتيح للتنقل بين الصفحات بسلاسة (سهم اليسار للتقدم، سهم اليمين للرجوع - متوافق مع اتجاه القراءة العربي).
- **الانتقال المباشر:** شريط تحكم عائم ذكي يتيح لك كتابة رقم الصفحة والقفز إليها فوراً.
- **تصور البيانات (Data Overlay):** زر مخصص لجلب البيانات من ملفات الـ JSON ورسم إحداثيات الآيات (مربعات برتقالية) وعلامات الوقف (دوائر زرقاء) فوق الصورة، مما يسهل عملية التدقيق والتصحيح.
- **ذاكرة محلية:** يحفظ العارض آخر صفحة قمت بزيارتها وحالة التحديدات (مفعلة/معطلة) ليعود إليها تلقائياً في جلستك القادمة.

---

## 🚀 كيفية التشغيل (هام)

بما أن العارض يقوم بجلب ملفات الـ `JSON` من المجلدات المجاورة (`databases` و `pages`)، فإن متصفحات الويب الحديثة تمنع ذلك لدواعي أمنية (CORS) إذا تم فتح ملف `index.html` مباشرة (بحيث يظهر المسار كـ `file:///`).

لتعمل الأداة بنجاح، يجب تشغيل خادم محلي (Local Server) من **الجذر الرئيسي للمشروع** (أي المجلد الأب `warsh-muthamma-assets`).

### الخيار الأول: باستخدام Node.js (مستحسن)
افتح سطر الأوامر (Terminal) في المجلد الرئيسي للمشروع `warsh-muthamma-assets`، ونفذ الأمر التالي:
```bash
npx http-server -p 8081 --cors
```

### الخيار الثاني: باستخدام Python
إذا كان لديك بايثون مثبت على جهازك، افتح سطر الأوامر في المجلد الرئيسي للمشروع ونفذ:
```bash
python -m http.server 8081
```

### 🌐 تصفح العارض
بمجرد تشغيل الخادم، يمكنك الوصول إلى العارض عبر فتح الرابط التالي في أي متصفح:
[http://localhost:8081/viewer/](http://localhost:8081/viewer/)
