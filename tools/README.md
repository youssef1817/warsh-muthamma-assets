# أدوات التحضير

هذا المجلد مخصص لاحقا لسكربتات التحويل والتحقق، مثل:

- استخراج الصفحات من PDF إلى صور خام.
- ضغط الصور وتحسين الحجم دون إفساد النص.
- توليد `manifest.json` بعد حساب الأحجام وهاشات الملفات.
- التحقق من أن الصفحات من `page001.png` إلى `page485.png` موجودة بلا نقص.

لم نضف سكربت التحويل بعد لأن أدوات مثل Poppler أو ImageMagick غير مثبتة حاليا في PATH على هذا الجهاز.
