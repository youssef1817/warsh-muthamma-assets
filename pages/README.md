# صور الصفحات

سيتم وضع الصور النهائية هنا بعد تحويل ملف PDF.

النمط المطلوب للتوافق مع منطق الطبعات المصورة الحالية في التطبيق:

```text
png/page001.png
png/page002.png
...
png/page485.png
```

يفضل البدء باستخراج خام عالي الجودة، ثم إنتاج نسخة PNG محسنة ومضغوطة بدون تغيير شكل الخط أو ألوان المصحف بقوة.
