# الحزم المضغوطة

هذا المجلد مخصص لحزم الصور التي سيحملها التطبيق.

الاسم المقترح للحزمة النهائية:

```text
warsh_muthamma_pages_png.zip
```

ينبغي أن تحتوي الحزمة على الملفات مباشرة أو ضمن مسار واضح يحافظ على الأسماء:

```text
page001.png
page002.png
...
page485.png
```

سنتأكد من البنية الدقيقة قبل دمجها في التطبيق حتى لا يحتاج كود التنزيل إلى معالجة خاصة.
