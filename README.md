# مصحف ورش المثمن

هذا المستودع مخصص لحفظ طبعة مصورة لورش المثمن، أي طبعة يكون فيها كل ثمن في صفحة غالبا، مع استثناءات في أوائل المصحف وأواخره.

الغرض منه أمران:

- توفير صور الصفحات الجاهزة للنشر والتنزيل المباشر.
- توفير بيانات مساعدة للتطبيق مثل توزيع الأثمان وبيانات تحديد الآيات.

## المحتويات الحالية

- `source/warsh-muthamma-source.pdf`: ملف الـ PDF الأصلي الذي استخرجت منه الصور.
- `pages/warsh_muthamma_png/page001.png` إلى `pages/warsh_muthamma_png/page485.png`: صور الصفحات النهائية.
- `pages/warsh_muthamma_png_index.json`: فهرس الصفحات النهائية مع الأحجام والأبعاد والبصمات.
- `data/thumn/warsh_muthamma_thumn_pages.json`: بيانات توزيع الأثمان على الصفحات.
- `manifest.json`: وصف تقني مختصر للطبعة وطريقة نشرها.
- `databases/ayahinfo/warsh_muthamma/`: بيانات التحديد الأولية الخاصة بالطبعة.

## ملاحظات مهمة

- عدد صفحات هذه الطبعة `485` صفحة.
- الصفحات تنشر وتحمّل مباشرة كملفات PNG منفصلة. لا توجد حزمة ZIP للصفحات في المسار الإنتاجي.
- ما يزال `ayahinfo_muthamma.zip` موجودا لأنه مطلوب لتحميل بيانات تحديد الآيات داخل التطبيق، وهو مختلف عن فكرة ضغط الصفحات نفسها.
- طبعة `warsh_muthamma` داخل التطبيق تعيد استخدام `quran.ar.warsh.db` الأصلية إذا كانت موجودة، أو يحمّلها التطبيق من مسار ورش الأصلي عند الحاجة.

## نمط النشر المعتمد

المسار المعتمد للصفحات هو التنزيل المباشر من GitHub Raw:

```text
pages/warsh_muthamma_png/page001.png
pages/warsh_muthamma_png/page002.png
...
pages/warsh_muthamma_png/page485.png
```

أما بيانات تحديد الآيات فتُقدَّم عبر:

```text
databases/ayahinfo/warsh_muthamma/ayahinfo_muthamma.zip
```

## تنبيه حقوقي

هذا المستودع لا يدعي ملكية النص القرآني ولا ملف الـ PDF الأصلي. الغرض هو خدمة القراءة والتيسير على المستخدمين. يرجى مراجعة وضع الحقوق المتعلق بالنسخة المصدر قبل أي استعمال تجاري أو إعادة نشر واسعة.
