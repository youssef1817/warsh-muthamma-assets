# مصحف ورش المثمن

هذا المستودع مخصص لحفظ نسخة مصورة من مصحف ورش المثمن، أي نسخة يكون فيها كل ثمن في صفحة في الغالب، مع استثناءات في أوائل المصحف وأواخره.

الغرض منه أن يكون مصدرا منظما للتحميل لمن أراد النسخة، وأن يوفر في الوقت نفسه ملفات يمكن للتطبيقات استعمالها مباشرة: صور الصفحات، حزم مضغوطة، وبيانات التقسيمات.

## المحتويات الحالية

- `source/warsh-muthamma-source.pdf`: ملف PDF الأصلي الذي استخرجت منه الصور.
- `data/thumn/warsh_muthamma_thumn_pages.json`: بيانات توزيع الأثمان على صفحات هذا الملف.
- `data/thumn/warsh_muthamma_thumn_pages.zip`: نفس بيانات توزيع الأثمان بصيغة مضغوطة.
- `manifest.json`: وصف أولي للنسخة والملفات المهمة.
- `pages/warsh_muthamma_png/page001.png` إلى `pages/warsh_muthamma_png/page485.png`: صور الصفحات الجاهزة بعد القص والضغط اللوني.
- `zips/warsh_muthamma_pages_png.zip`: الحزمة المضغوطة الجاهزة للرفع.

## ملاحظات مهمة

- عدد صفحات هذه النسخة 485 صفحة.
- بيانات تحديد مواضع الآيات غير مكتملة حاليا. يمكن استعمال النسخة للعرض والتصفح، أما تحديد الآية بدقة فوق الصورة فيحتاج عملا مستقلا.
- لا يدعي هذا المستودع ملكية النص القرآني. الغرض هو خدمة القراءة والتيسير على المستخدمين. يرجى مراجعة حقوق ملف PDF أو الجهة الناشرة قبل أي استعمال تجاري أو إعادة نشر واسعة.

## بنية التحميل المقترحة للتطبيق

عند رفع هذا المستودع إلى GitHub يمكن استعمال روابط raw أو GitHub Releases. الأفضل للصور الكبيرة أن توضع الحزم في Releases أو في تخزين خارجي سريع، مع إبقاء `manifest.json` في المستودع.

النمط الحالي للصور:

```text
pages/warsh_muthamma_png/page001.png
pages/warsh_muthamma_png/page002.png
...
pages/warsh_muthamma_png/page485.png
```

والحزمة الجاهزة:

```text
zips/warsh_muthamma_pages_png.zip
```
